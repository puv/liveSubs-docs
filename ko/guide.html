브라우저 애플리케이션 ​

웹앱 사용법 ​

OBS는 창이 최소화된 경우 자막을 캡처하지 않습니다.
이 문제를 해결하려면 liveSubs의 앱 버전을 사용하세요.

  1. https://subs.puv.bar로 이동합니다.
  2. 왼쪽 상단의 '허용'을 클릭하여 마이크 사용을 허용하세요.
  3. 말할 언어를 선택하세요. '자막' 행의 '언어' 열에서 선택할 수 있습니다.
  4. 번역 추가를 클릭하여 번역 옵션을 추가하세요. 최대 5개의 번역 옵션을 추가할 수 있습니다.
    1. 각 번역 옵션의 언어를 선택하세요
  5. 설정 변경이 완료되면 '닫기' 버튼을 클릭하세요.

다음 -> OBS 설정

웹앱 OBS 설정 ​

  1. 선택한 녹음 소프트웨어에서 새로운 '창 캡처' 소스를 추가하세요.
  2. 'liveSubs'라는 제목의 창을 선택하고 '확인'을 클릭하세요.
  3. '확인'을 클릭하세요.

Windows 애플리케이션 ​

앱 설치 ​

앱이 서명되지 않았기 때문에 브라우저나 Windows에서 경고를 받을 수 있습니다.
이 애플리케이션은 안전하게 사용할 수 있으며, 소스 코드는 여기에서 확인할 수 있습니다.
``안전한 것으로 신고하기''를 눌러주시면 프로젝트에 도움이 되실 것 같아 정말 좋을 것 같습니다.

  1. https://github.com/puv/liveSubs/releases/latest로 이동합니다.
  2. liveSubs의 '최신 버전'을 다운로드하세요.
    1. 브라우저에서 경고 메시지가 나타나면 '...' 또는 '더 보기'를 클릭한 후 '유지'를 클릭하세요.
    2. 확인 메시지가 나타나면 '어쨌든 유지'를 클릭하세요.
  3. 다운로드한 파일을 엽니다.
    1. Windows에서 경고 메시지가 나타나면 '추가 정보'를 클릭한 다음 '어쨌든 실행'을 클릭하세요.

다음 -> Windows 애플리케이션 사용

Win-App 사용법 ​

  1. 다운로드한 파일을 열어주세요
  2. 왼쪽 상단의 '허용'을 클릭하여 마이크 사용을 허용하세요.
  3. 말할 언어를 선택하세요. '자막' 행의 '언어' 열에서 선택할 수 있습니다.
  4. 번역 추가를 클릭하여 번역 옵션을 추가하세요. 최대 5개의 번역 옵션을 추가할 수 있습니다.
    1. 각 번역 옵션의 언어를 선택하세요
  5. 설정 변경이 완료되면 '닫기' 버튼을 클릭하세요.

다음 -> OBS 설정

Win-App OBS 설정 ​

  1. 선택한 녹음 소프트웨어에서 새로운 '브라우저 소스' 소스를 추가하세요.
  2. URL에 https://subs.puv.bar/client를 입력하고 닫기를 클릭하세요.
  3. '확인'을 클릭하세요.

다음 -> 배경 제거

배경 제거 ​

  1. 방금 추가한 Window Capture 소스를 마우스 오른쪽 버튼으로 클릭하고 필터 -> 필터 편집을 클릭합니다.
  2. '+' 버튼을 클릭하고 '크로마 키'를 선택한 후 '추가'를 클릭하세요.
  3. 색상 선택기를 클릭하고 자막 배경 색상을 선택하세요. 기본적으로 녹색입니다.
  4. '닫기'를 클릭하세요.
